Twitticious allows you to import links from a twitter timeline to your del.icio.us account.

Twitticious doesn't work yet on shared webhost, you can download the project and use it from your computer.

  1. Download project on github
  2. launch app: ruby twitticious.rb
  3. Access to the app from your web browser: http://localhost:4567

Gem requirement:

  • Sinatra
  • rDelicious

Project on GitHub: Twitticious
